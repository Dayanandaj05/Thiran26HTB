## Setup Instructions

1. Copy challenge folders to your web server directory
2. Configure CTFd with the above challenge details
3. Update [YOUR_LOCAL_SERVER] with your actual server IP/hostname
4. Test each challenge before the event
5. Ensure flag validation is case-sensitive
